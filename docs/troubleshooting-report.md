# Troubleshooting Report

## Issue 1 — Incorrect IP Configuration

### Symptoms
- Ping request timed out
- Devices unable to communicate

### Cause
- PC2 configured with wrong subnet

### Solution
- Reconfigured IP address

### Result
- Successful ping response restored

## Issue 2 — DNS Misconfiguration

### Symptoms
- Browser unable to access websites

### Cause
- Incorrect DNS server configured

### Solution
- Changed DNS to 8.8.8.8

### Result
- Internet access restored
